#                                            大数据学习之shell

### 第一章shell 概述

- shell 是一个命令行解释器，它接受应用程序/用户命令程序，然后在调用回内核
- shell 还是一个功能强大的编程语言，易编写，易调试，灵活性强。

### 第二章shell脚本入门

- 脚本格式

```shell
#！/bin/bash
```

- 脚本的常用执行方式

  ```shell
  1.bash 脚本路径
  2.脚本路径下：./脚本名
  ```
